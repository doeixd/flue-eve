# Packages

| Import path | Purpose |
|-------------|---------|
| `flue-eve` | Public package (future single install) |
| `flue-eve/vite` | Vite plugin for development |
| `flue-eve/client` | Eve-compatible client SDK |
| `flue-eve/react` | `useEveAgent` React hook |
| `flue-eve/config` | `defineConfig` for `eve.config.ts` |
| `@flue-eve/compat-server` | Server-side compat module |

Currently, installation requires individual internal packages.
The single public package is planned — see [public package](/docs/packages/public-package).
