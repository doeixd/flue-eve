# Vite Plugin

```ts
import { flueEve } from 'flue-eve/vite';
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `agentName` | string | `'assistant'` | Default agent name |
| `scaffold` | boolean | `false` | Scaffold agent files on first run |
| `spawnFlueDev` | boolean | `true` | Auto-start `flue dev` server |
| `flueBaseUrl` | string | — | Flue server URL for proxy |
| `proxyMode` | string | `'eve-routes-only'` | Proxy mode for split-origin |
