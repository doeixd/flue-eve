# Migration Support

Migration tooling is planned to help users move from an existing Eve
project to flue-eve:

- Import alias scanner (detects `eve/*` imports and suggests replacements)
- Config converter (Eve config → flue-eve config)
- Agent file migrator (Eve file layout → Flue `createAgent` calls)

**Status:** Planned. Track on the [roadmap](/docs/roadmap/full-compatibility).
