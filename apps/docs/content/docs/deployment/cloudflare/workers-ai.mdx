# Workers AI

Workers AI support is planned. When available, you will configure the
model in the Flue agent definition:

```ts
createAgent({
  model: 'workers-ai/@cf/meta/llama-3.1-8b-instruct',
});
```

See [roadmap](/docs/roadmap/full-compatibility) for status.
