# Environment Variables

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `FLUE_BASE_URL` | No | — | Flue server URL for loopback admission |
| `FLUE_AGENT_NAME` | No | `assistant` | Default agent name |
| `FLUE_EVE_MOUNT` | No | `/eve/v1` | Route mount path |
| `EVE_AUTH_TOKEN` | No | — | Bearer token for auth mode |
| `EVE_PERSISTENCE` | No | `memory` | Persistence adapter type |

See the individual deployment pages for framework-specific variables.
