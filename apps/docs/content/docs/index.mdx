# flue-eve

**Eve authoring + frontend/client contract on an open Flue runtime.**

`flue-eve` is a compatibility layer that lets you use Eve-style HTTP routes,
client SDKs, and React hooks while running your agents on
[Flue](https://flueframework.com) — an open, deploy-anywhere agent runtime.

## What you get

- **Eve HTTP routes** — `/eve/v1/health`, `/session`, `/stream` work as documented
- **Eve client SDK** — drop-in `eve/client` or use `flue-eve/client`
- **`useEveAgent` hook** — drop-in from `eve/react` or use `flue-eve/react`
- **Vite plugin** — proxy / scaffold / aliases in one dependency
- **Flue runtime** — agents, tools, durable execution, deploy on Node or Cloudflare
