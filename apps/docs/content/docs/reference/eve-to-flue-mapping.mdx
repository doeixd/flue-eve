# Eve-to-Flue Mapping

| Eve concept | Flue equivalent | Notes |
|-------------|-----------------|-------|
| Session | Agent instance | `sessionId` = Flue instance `id` |
| Turn | `POST /agents/:name/:id` | Admission sends message to Flue |
| Stream | Durable Streams | Translated to NDJSON by mapper |
| Token | Adapter-issued | Stable per-session in v1 |
| Agent file layout | `createAgent` config | Scaffolded from Eve layout via `npx flue-eve init` |
| Channel | Flue channel event | Bridged by `@flue-eve/channels` |
| Workflow | Flue workflow run | Bridged by `@flue-eve/workflows` |
