# Migration Tiers

| Tier | Description | Status |
|------|-------------|--------|
| 0 | HTTP routes + client SDK + React hook | **Supported** (M0–M4) |
| 1 | HITL + tool calls + connections | **Supported** (M5) |
| 2 | Production auth + persistence + Cloudflare | **Supported** (M6) |
| 3 | Authoring model (Eve file layout) | **Partial** (M7 scaffold) |
| 4 | Workflows + channels | **Supported** (M8e) |
| 5 | SvelteKit + Nuxt + Nitro plugins | **Supported** (M8f–M8g) |

See [full compatibility](/docs/reference/compatibility) for the complete
surface-by-surface status.
